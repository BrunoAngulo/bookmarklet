Funciones del equipo Ted

Herramientas listas para arrastrar a tu barra de favoritos. Sin instalaciones, sin extensiones: solo arrastra y úsalas.

Conexión con Jira

Sin token

El registro de worklog usa la API de Jira con tu correo y un API token. Se guarda solo en este navegador y se incrusta en el favorito que arrastres. Genera tu token en id.atlassian.com.

El token se guarda localmente para generar los favoritos.
⏱️

Registro de Worklog en Jira

Detecta automáticamente cuando agregas un comentario en un ticket de Implementación Colegio o Análisis pseudo integración Tangerine, y abre un formulario para registrar la actividad y el tiempo invertido directamente en el worklog del ticket.

Jira Cloud Worklog estructurado Detección de comentarios
⏱️ Registrar Worklog Ted

Guarda primero tu API token para activar el favorito.

Cómo se usa
  1. Guarda tu correo y API token de Jira arriba.
  2. Arrastra Registrar Worklog Ted a la barra de favoritos.
  3. Entra al ticket en project-tools-santillana.atlassian.net y haz clic en el favorito una vez para activarlo.
  4. Escribe y envía un comentario. Si el título empieza por Implementación Colegio: o Análisis pseudo integración Tangerine, aparece el formulario.
  5. El tiempo se pre-llena con lo transcurrido desde que activaste el detector (formato 7w 2d 1h 30m, editable). Elige la actividad y pulsa Guardar.
🎓

Publicar Tangerine en Classroom

Dentro de Tangerine, agrega la opción Compartir en class al menú de cada recurso para publicarlo en Google Classroom con un mensaje editable. Requiere conectar Google primero.
